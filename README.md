# Kanon

Eine Sammlung deutscher Literatur-Bestenlisten als strukturierte Daten.

## Enthaltene Listen

| Liste | Zeitraum | Einträge |
|-------|----------|----------|
| ZEIT 100 beste Bücher | 2020-2025 | ~600 |
| Reich-Ranicki Kanon | 2002 | 150 |
| Deutscher Buchpreis | 2005-2025 | ~400 |

## Nutzung

Die Daten liegen als JSON vor und folgen einem einheitlichen Schema.
Gut geeignet für:
- Leseempfehlungen
- Literaturwissenschaftliche Analysen
- Visualisierungen
- Integration in Lese-Apps
